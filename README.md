# ghst Agent Skill for managing the Ghost CMS

**Version: 1.0.3**

This repository contains the `ghst` skill for Agents, allowing your AI agent to interact directly with any Ghost blog or publication via the Ghost Admin API. 
It acts as a wrapper around the [@tryghost/ghst](https://github.com/TryGhost/ghst) CLI tool.

It's available from:
- [https://github.com/nickludlam/ghst-for-ghost-skill](https://github.com/nickludlam/ghst-for-ghost-skill)
- [https://clawhub.ai/nickludlam/ghst](https://clawhub.ai/nickludlam/ghst)

## Usage

Once installed and configured, your agent can manage posts, pages, members, tags, newsletters, themes, and more on your Ghost instance.

For full details on agent capabilities, command references, and specific workflows, please see the primary [SKILL.md](SKILL.md) instruction file.
